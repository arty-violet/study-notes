append и appendChild

С помощью метода append и appendChild можно добавлять в конец родительского элемента дополнительные элементы либо содержимое для элемента

элемент-родитель.append('добавляемый-элемент');
элемент-родитель.append('какой-либо текст');

элемент-родитель.appendChild('добавляемый-элемент');
элемент-родитель.appendChild('какой-либо текст');

* Добавлять с помощью этих методов можно и элементы, и простые строки.

createElement

~ Создание DOM-элемента ~

С помощью метода createElement можно создать элементный узел внутри DOM структуры 

document.createElement('имя тега');

Метод принимает на вход строку с именем тега и возвращает созданный DOM-элемент. Например создадим элемент <div>:

let newElement = document.createElement('div');
// Будет создан новый элемент <div> и записан в переменную.

~ Добавление свойств ~

Для изменения свойств созданного элемента можно воспользоваться дополнительными свойствами, например: textContent - для замены содержимого элемента, либо classList с методом add - для добавления класса новому элементу.

newElement.textContent = 'Я новый абзац!';
newElement.classList.add('some-class');

~ Добавление в DOM-дерево ~

Вновь созданный элемент по умолчанию не находится в DOM-дереве и не отображается на странице. Мы просто храним его где-то в переменной. Чтобы новый элемент появился на странице, его нужно добавить в DOM. Для этого найдём в текущем DOM-дереве элемент (родитель) и вставим в него наш элемент.

let parent = document.querySelector('.parent');

Чтобы добавить новый элемент в конец элемента-родителя, можно воспользоваться методом append либо appendChild

parent.append(newElement);
parent.appendChild(newElement);

classList.contains

С помощью свойства classList с применением к нему метода contains можно проверять наличие класса

element.classList.contains('class');

* Метод classList.contains вернёт true(истина), если класс у элемента есть, и false(ложь), если класса нет.

classList.add

С помощью свойства classList с применением к нему метода add можно добавлять класс элементу

element.classList.add('сlass');

classList.remove

С помощью свойства classList с применением к нему метода remove можно убирать класс у элемента

element.classList.remove('сlass');

classList.toggle

С помощью свойства classList с применением к нему метода toggle можно переключать классы у элемента

element.classList.toggle('сlass');
example code:
let element = document.querySelector('.сlass');
if (element.classList.contains('some-class')) {
  console.log('Условие истинно');
  // Сообщение выведется, только если класс у элемента есть (возвращает true).
}

querySelector

С помощью метода querySelector можно находить элементы

document.querySelector('selector');

* document - элемент, внутри которого будет искать JavaScript. Вместо элемента document можно использовать и любые другие родительские элементы.

* querySelector - метод или то, что нужно сделать.

querySelectorAll

С помощью метода querySelectorAll можно находить все элементы (коллекцию)

* DOM-коллекция, полученная через querySelectorAll похожа на массив, однако им не является и название таких коллекций — псевдомассив.

let elements = document.querySelectorAll('selector');

* Коллекцию, как и обычный элемент, можно сохранить в переменной.

Самый простой способ узнать, какие элементы содержит коллекция, — вывести её в консоль:

console.log(elements);

К элементам коллекции можно обращаться по индексу

Индекс — это порядковый номер элемента в коллекции.

* !! Отсчёт начинается с нуля, поэтому у первого элемента индекс 0, а у второго — 1.

коллекция[индекс] ~ elements[i]
let elements = document.querySelectorAll('selector');
console.log(elements[0]); // Выведет первый элемент коллекции
console.log(elements[1]); // Выведет второй элемент коллекции

К элементам коллекции выбранным по индексу можно добавлять обработчики событий

elements[0].событие = function () {};
elements[1].событие = function () {};

scrollTo

Чтобы прокрутить страницу используем метод scrollTo:

window.scrollTo(координата - Х, координата - Y);

Координата Х указывает, куда нужно прокрутить страницу по горизонтали, а координата Y— куда нужно прокрутить страницу по вертикали.

Точка указывается относительно левого верхнего угла окна.

window.scrollTo(100, 50); // Прокрутит страницу на 100px вправо и на 50px вниз

* Координаты задаются в пикселях, но указывать единицы измерения не нужно

Чтобы при взаимодействии с элементом, например при клике на него, страница прокручивалась к началу, нужно добавить инструкцию с методом scrollTo внутрь обработчика событий onclick.

элемент.onclick = function () {
  window.scrollTo(0, 0);
};

toString()

Чтобы привести числовое значение к строковому типу. Один из способов — использовать метод toString(). Например:

var number = 1;
console.log(number); // выведет число: 1 (number)
console.log(number.toString()); // выведет строку: 1 (string)

parseInt()

Чтобы превратить строковое значение к числовому типу. Один из способов — использовать метод parseInt(). Например:

var string = '1';
console.log(string); // выведет строку: 1 (string)
console.log(parseInt(string, 10)); // выведет число: 1 (number)
