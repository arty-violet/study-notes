"Основы JavaScript"

----------------------------------

+-

Основные понятия

JavaScript особым образом воспринимает разметку: элементы здесь не строки, которые мы пишем в HTML-файлах, а объекты. Каждый объект связан с другими такими же объектами и знает о своём родителе, соседних объектах-элементах, вложенных объектах. В итоге получается древовидная структура, которая называется DOM (Document Object Model или объектная модель документа).

В каждом DOM-дереве есть корневой объект, из которого «растут» другие объекты. Он называется document. document — страница, которая содержит все элементы разметки (объекты).

Подключение скриптов

Скрипты подключаются к странице с помощью тега script двумя способами:

Инлайновый код

Код пишется внутри тега script.

<body> 
  …
  <script>console.log('Я — инлайновый скрипт');</script> 
</body>

Внешний файл с кодом

Тегу script добавляется атрибут src, в котором указывается путь до файла со скриптом.

<body> 
  …
  <script src="script.js"></script> 
</body>

Вывод элемента в консоль

console.log('Привет от JavaScript!');
console.log(document.querySelector('selector'));

// Вывод массива в консоль
console.log([1, 2, 3, 4, 5]);

// Вывод объекта в консоль
console.log({
  month: 'june',
  day: 15
});

* Консоль — это инструмент разработчика, который помогает тестировать код.

Типы данных

number — числа;
string — строки;
boolean — логические или булевы значения, «истина (true)» или «ложь (false)»;

* Все числа кроме 0 — true, т.е. 0 — false.

* Все строки кроме пустой строки — true, т.е. пустая строка '' — false.

array — массив, хранит последовательность значений и порядок этих значений важен;
object — объект, состоит из множества пар «ключ-значение», порядок этих пар не важен. 

// Массив
[1, 2, 3, 4, 5]

// Объект
{
  month: 'june',
  day: 15
}

Объявление и присваивание переменных

Для объявления переменных используются заявления var, let и const после которых необходимо объявить переменную

Для операции присваивания используется знак равенства

let number = 7;
let variableName = 'Я значение переменной!';
let isElementTrue = true;
let element = document.querySelector('selector');

* Зарезервированные ключевые слова в ECMAScript 2015:https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Lexical_grammar#Ключевые_слова

Функции

Функция - это блок кода, предназначенный для выполнения конкретной задачи. Функцию можно использовать многократно написав всего один раз. Она не просто содержит в себе значение, как переменная, а выполняет какое-то действие и решает какую-то задачу: считает, сравнивает, ищет.

var functionName = function (parameter1, parameter2) {
  // Действия, которые выполняет функция или «тело функции»
};

functionName — имя переменной, которая содержит в себе функцию. По этому имени мы сможем обращаться к функции и использовать её много раз.

* Название такой переменной это и есть название функции.

* Название функции должно быть в форме глагола - это общепринятое правило именования.

Чтобы создать функцию используем ключевое слово function, а дальше внутри круглых скобок добавляем параметры, которые функция будет использовать каждый раз, как мы к ней обратимся.

* Параметры работают так же, как переменные.

~ Вызов функции ~

Чтобы функция начала свою работу, нужно обратиться к функции по её имени, с указанными параметрами (аргументами) внутри круглых скобок.

functionName(parameter1, parameter2);

* Кстати, правильно говорить «функция принимает параметры», но при этом мы «передаём функции аргументы». Дело в том, что параметры - значения, которые мы задаём в момент объявления функции. Их же мы используем и в теле функции. А аргументы - значения, которые мы передаём в функцию при её вызове.

~ Возврат функции ~

Чтобы функция смогла выполнить код и отдать (возвратить) результат операций для дальнейшей работы с этим результатом используется оператор return

* Код, написанный на новой строке после return, не выполняется.

* Функция не может вернуть сразу много значений, она возвращает только один результат.

* Если внутри функции нет return или после return не указано, какое значение нужно вернуть, функция вернёт undefined, иными словами, ничего.

var functionName = function (parameter1, parameter2) {
  var result = parameter1 + parameter2;
  return result;
};

~ Функции как переменные ~

Функции, которые имеют собственное название можно использовать так же, как и переменные.

var variableName = functionName(parameter1, parameter2);

~ Анонимные функции ~

Функции, которые не имеют названия, называются анонимными функциями.

function () {};

~ Локальные переменные ~

Переменные объявленные внутри функции, являются локальными и могут использоваться только внутри функции.

// переменная localVariable в этой части кода недоступна

var functionName = function (parameter1, parameter2) {
  var localVariable = "значение переменной";
  // переменная localVariable может доступна только внутри этой функции
}

// переменная localVariable в этой части кода недоступна

* Поскольку локальные переменные распознаются только внутри их функций, переменные с одинаковым именем могут использоваться в разных функциях.

То же самое произойдёт, если мы захотим обратиться снаружи к параметру функции. Параметр хоть и задаётся снаружи, но ведёт себя, как переменная внутри функции.

А всё из-за того, что у каждой функции есть область видимости — все значения, доступные для этой функции. Область видимости ограничивается самой функцией.

~ Замыкания ~

Замыкание — функция, которая помнит о своём окружении. То есть это функция + все значения переменных вне локальной области видимости этой функции.

* Речь идёт только о переменных, которые функция использует в своём коде.

Благодаря замыканиям мы можем зафиксировать какое-то значение в функции, а использовать саму функцию позже.

var functionName = function (parameter) {
  return function () {
    console.log(parameter);
  };
};
 
var someVariable = functionName('любое значение параметра');
schoolkid(); // Выведет: любое значение параметра

Объект event

Объект event — параметр функции-обработчика. Он всегда передаётся браузером в эту функцию в момент наступления события.

Чтобы использовать event, достаточно указать этот объект параметром функции-обработчика и написать инструкции.

У объекта события есть много полезных свойств. Например, тип события, введённый символ, координаты курсора в момент наступления события и многое другое.

preventDefault()

Чтобы отменить для некоторых элементов страницы действие по умолчанию, например для кнопок с типом submit или для ссылок с указанным href применяется метод preventDefault().

Для этого достаточно к объекту event, применить метод preventDefault().

link.addEventListener('click', function(evt) { 
  evt.preventDefault(); // Отменяем действие по умолчанию
  // Добавляем инструкции для события клика
});

keyCode

С помощью свойства keyCode для каждой клавиши, которую нажал пользователь можно определить её код. Найти код любой клавиши можно здесь.

document.addEventListener('keydown', function(evt) {
 if (evt.keyCode === 27) { // Проверяем, что код клавиши равен 27
   // Код отсюда выполнится только при нажатии ESC
 }
});

* Кроме keyCode есть и другие свойства для определения нажатой клавиши. Например, key и code. Они возвращают названия клавиш, а не их номера.
