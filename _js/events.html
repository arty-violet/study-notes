"События"

----------------------------------

+-

Что такое обработчик событий

Действие, когда пользователь что-то делает с элементами страницы (например, кликает по кнопке или нажимает клавишу) называется событием. 

element.onclick = function () {
  console.log('Клик!');
};

* Клик по элементу - это событие, а та часть инструкции, которая идёт после onclick, называется обработчиком событий.

Обработчик события для элемента можно добавить ещё и другим способом с помощью функции addEventListener() ~ «слушатель события». 

element.addEventListener('click', function () {
  // Инструкции, которые будут выполняться при клике на элемент
});

Первый параметр функции addEventListener - название события, которое мы хотим поймать.

* Название записывается строкой, поэтому обязательно указывается в кавычках. Для всех событий есть специальные зафиксированные названия, список которых можно посмотреть здесь.

Второй параметр addEventListener — функция. Это и есть обработчик события.

* Внутри этой функции мы указываем, что нужно сделать, когда произойдёт событие, как его нужно обработать. То есть действия из тела этой функции выполнятся только тогда, когда произойдёт установленное событие.

Из этого также следует, что события асинхронны, то есть происходят не по расписанию в какое-то конкретное время, друг за другом, а тогда, когда понадобится.

keydown

Событие keydown срабатывает при нажатии на любую клавишу

* Слушать это событие можно только на элементах, которые имеют состояние фокуса: поля ввода, кнопки, элементы с атрибутом tabindex, документ. При нажатии фокус должен находиться на соответствующем элементе.

document.addEventListener('keydown', function() {
  // Код отсюда выполнится при каждом нажатии любой клавиши
});

* onchange срабатывает, если значение поля ввода изменилось и пользователь закончил ввод. Например, если пользователь передвинул ползунок и отпустил его. Или ввёл что-то в текстовое поле и убрал из него курсор.

onchange

Обработчик событий onchange используется для того, чтобы отследить событие об изменении элемента, например когда пользователь выбирает новое значение из выпадающего списка, а также при переключении чекбокса или радиокнопки

select.onchange = function () {
  console.log(select.value); // Выводим в консоль новое значение
};

checkbox.onchange = function () {
  // Инструкции выполняемые при переключении чекбокса
};

* onchange срабатывает, если значение поля ввода изменилось и пользователь закончил ввод. Например, если пользователь передвинул ползунок и отпустил его. Или ввёл что-то в текстовое поле и убрал из него курсор.

oninput

Получить текстовые значения можно с помощью обработчика событий oninput.

element.oninput = function () {};

Инструкции внутри обработчика oninput выполняются каждый раз, когда значение в поле ввода меняется

Благодаря обработчику событий oninput мы можем получить текстовые значения сразу во время набора текста, цифр или символов

* oninput срабатывает на каждое изменение значения, независимо от того, завершил пользователь ввод или нет. Например, он сработает на каждое изменение положения ползунка, даже если пользователь продолжает его двигать. И на каждый новый символ в текстовом поле, даже если пользователь продолжает вводить текст.

onsubmit

С помощью обработчика события onsubmit можно отправить форму

element(form).onsubmit = function (evt) {
  evt.preventDefault();
  console.log('Форма отправлена!');
};

!! Обработчик событий onsubmit добавляется не на кнопку, а на саму форму.

* Инструкция evt.preventDefault(); отменяет отправку данных.

onscroll

Событие onscroll происходит, когда полоса прокрутки элемента прокручивается.

object.onscroll = function () {
 console.log('Страница прокручена');
}

* window - окно (или вкладка) браузера

* Обработчик событий onscroll можно добавлять не только окну браузера, но и отдельным элементам на странице.

Чтобы показать елемент при прокрутке, нужно внутрь обработчика событий onscroll добавить условную конструкцию и с помощью оператора сравнения сравнить величину прокрутки с числом, например с числом 200 равное 200px:

if (window.pageYOffset > 200) {

* Если условие истинно, показываем элемент

}
