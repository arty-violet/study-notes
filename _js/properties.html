"Свойства"

----------------------------------

+-

alt

С помощью свойства alt можно задать изображению альтернативный текст, то есть описание фотографии.

picture.alt = 'описание изображения';

children

Свойство children возвращает ращает коллекцию дочерних, то есть вложенных, DOM-элементов.

element.children

classList

Свойство classList возвращает имена классов для элемента в виде объекта DOMTokenList.

Это свойство полезно для добавления, удаления и переключения классов CSS в элементе.

Свойство classList доступно только для чтения, однако вы можете изменить его, используя методы add (), remove (), toggle ().

element.classList

* DOMTokenList - список содержащий имена классов элемента

dataset

Атрибуты, начинающиеся с префикса data-, обычно используют, чтобы хранить вспомогательную информацию. Получить значение data-атрибута можно с помощью свойства dataset.

element.dataset.attributeName ~ element.dataset.имяАтрибутаБезПрефикса

* Если имя атрибута состояло из нескольких слов и в нём были дефисы, то в JavaScript его записывают в «верблюжьем» стиле (camelCase): дефисы убирают, а каждое слово, кроме первого, пишут с большой буквы.

console.log(element.dataset.attributeName); 
// выведет значение data-attribute-name

length

С помощью свойства length можно вычислить длину текстовой строки либо количество элементов в массиве. Значение свойства length равно числу символов в строке либо количеству элементов в массиве.

* Символами считаются не только буквы и цифры, но также пробелы и переносы строки, а длина пустой строки либо пустого массива равна 0.

string.length; // Возвращаем длину строки
array.length; // Возвращаем длину массива

checked

Чтобы проверить состояние элемента, например чекбокса, используем свойство checked.

Это свойство имеет булево значение: true, если чекбокс включён, и false, если нет.

if (checkbox.checked) {
  // Инструкции выполнятся, если чекбокс включён
} else {
  // Инструкции выполнятся, если чекбокс выключен
}

disabled

Заблокировать, а также снять блокировку с элемента можно с помощью свойства disabled.

element.disabled = true;

* Блокирует элемент

element.disabled = false;

* Разблокирует элемент

pageYOffset

Свойство pageYOffset содержит число пикселей, на которое пользователь прокрутил страницу по вертикали

console.log(window.pageYOffset); // Выведет количество пикселей

С помощью этого свойства можно например показать либо скрыть елемент в зависимости от прокрутки страницы. Для этого нужно к условной конструкции if добавить условие при котором необходимый элемент будет показываться, например при прокрутке страницы вниз более чем на 200px. И противоположное условие при котором элемент будет скрываться.

if (window.pageYOffset > 200) {
  // Если условие истинно, показываем элемент
} else {
  // Если условие ложно, скрываем элемент
}

style

С помощью свойства style можно управлять стилями элемента. После него через точку указывают то CSS-свойство, которое нужно изменить.

element.style.свойствоCSS

Чтобы изменить стиль элемента, указанному свойству нужно присвоить новое значение.element.style.color = 'green';

* В JavaScript нельзя использовать дефисы в названиях свойств, вместо этого для разделения слов используется «верблюжий» стиль.

src

С помощью свойства src можно задать картинке адрес файла с изображением, то есть значение атрибута src.

picture.src = 'images/picture.jpg';

type

Свойство type в JavaScript отвечает за тип применяемый в тегах input.

Чтобы изменить тип поля ввода, нужно записать в свойство type новое значение. Например сделаем input текстовым полем ввода:

input.type = 'text';

textContent

С помощью свойства textContent можно выявлять либо менять текстовое содержимое элемента

let element = document.querySelector('selector');
console.log(element.textContent); // Выведет: текст внутри элемента.
element.textContent = 'Новый текст, который появится внутри элемента!';

* Свойство textContent хранит в себе текстовое содержимое элемента.

С помощью пустой строки можем очистить текстовое содержимое элемента.

element.textContent = '';

value

С помощью свойства value можно получить данные из поля ввода.

let input = document.querySelector('input');
console.log(input.value);
input.value;

С помощью пустой строки можем очистить данные из поля ввода.

input.value = '';
