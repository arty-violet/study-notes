Арифметические операторы

Арифметические операторы используются для выполнения арифметических операций между переменными и(или) значениями.

'+' - сложение
'-' - вычитание
'*' - умножение
'/' - деление
'%' - деление без остатка
'++' - инкремент
'--' - декремент

Операторы присваивания

Операторы присваивания используются для присвоения значений переменным.

Таким образом с их помощью можно изменять числовое значение.

Чтобы увеличить или уменьшить число в JavaScript, можно использовать полную или краткую записи.

let number = 0;

Полная запись

number = number + 2; // Значение переменной: 2
number = number - 2; // Значение переменной: 0

Краткая запись

number += 2; // Значение переменной: 2
number -= 2; // Значение переменной: 0

Для увеличиния и уменьшения на единицу в JavaScript есть особая запись

Увеличит число на 1

number++; // Значение переменной: 1

Уменьшит число на 1

number--; // Значение переменной: 0

Строковые операторы (конкатенация)

Оператор '+' можно использовать для объединения (конкатенации) строк.

console.log('Конкатенация - это' + ' сложение строк');

let method = 'Конкатенация - это';
console.log(method + ' сложение строк');

Операторы сравнения

Операторы сравнения используются в логических выражениях для определения равенства или разницы между переменными или значениями

* Значения, которые проверяет оператор, называют операндами.

Разница межды операторами строгого и нестрогого равенства(неравенства) в том, что нестрогие операторы сравнения приводят строковое значение в числовое, а строгие операторы сравнения нет.

- Оператор нестрогого равенства

'123' == 123; // Результат: true

- Оператор строгого равенства

'123' === 123; // Результат: false

- Оператор нестрогого неравенства

'123' != 123; // Результат: false

- Оператор строгого неравенства

'123' !== 123; // Результат: true

* Оператор строгого неравенства работает противоположно оператору строгого равенства. Если значения равны, он вернёт false, а если не равны, то true.

- Операторы сравнения больше и меньше

1 > 2; // Оператор «больше»
1 < 2; // Оператор «меньше»
1 >= 2; // Оператор «больше или равно»
1 <= 2; // Оператор «меньше или равно»

Логические операторы

Логические операторы используются для определения логики между переменными или значениями.

- Логическиий оператор И

Если хоть один из операторов вернёт false, тогда условие целиком будет считаться ложным:

true && true; // Результат: true
true && false; // Результат: false
false && true; // Результат: false
false && false; // Результат: false

- Логический оператор ИЛИ

Если хоть один из операторов вернёт true, тогда условие целиком будет считаться истинным:

true || true; // Результат: true
true || false; // Результат: true
false || true; // Результат: true
false || false; // Результат: false

- Логический оператор НЕ

Чтобы создать проверки с отрицанием, используют унарный (одноместный) логический оператор '!':

var condition = false;
if (!condition) { // Если условие не истинно, т.е. равно false
  // код выполнится
}

Математический объект

Объект Math позволяет выполнять математические задачи над числами.

Math.round()

Math.round(x) возвращает значение x, округленное до ближайшего целого числа:

Math.round(4.7); // returns 5
Math.round(4.4); // returns 4

Math.pow()

Math.pow(x, y) возвращает значение x в степень y:

Math.pow(8, 2); // returns 64

Math.sqrt()

Math.sqrt(x) возвращает квадратный корень из x:

Math.sqrt(64); // returns 8

Math.abs()

Math.abs(x) возвращает абсолютное (положительное) значение x:

Math.abs(-4.7); // returns 4.7

Math.ceil()

Math.ceil(x) возвращает значение x, округленное до ближайшего целого числа:

Math.ceil(4.4); // returns 5

Math.floor()

Math.floor(x) возвращает значение х округляется вниз до ближайшего целого числа:

Math.floor(4.7); // returns 4

Math.min() и Math.max()

Math.min(n1, n2, n3, ..., nX) и Math.max(n1, n2, n3, ..., nX) может использоваться, чтобы найти самое низкое или самое высокое значение в списке аргументов:

Math.min(0, 150, 30, 20, -8, -200); // returns -200
Math.max(0, 150, 30, 20, -8, -200); // returns 150
